# Site operations registry

This directory is not published by Jekyll. It is the private source of truth for
the future site-ops dashboard.

Tracked files:

- `sites/*.json`: one manifest per managed site

Generated files:

- `managed-sites.json`: dashboard-ready registry generated by
  `scripts/generate_site_registry.py`

Generate the registry:

```bash
./scripts/generate_site_registry.py
```

The registry is intentionally ignored because it includes a generation timestamp.
